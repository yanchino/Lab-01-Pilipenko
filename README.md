# Лабораторная работа №1: Разработка структуры проекта

**Исполнитель:** Пилипенко Д.А.  
**Группа:** ИС-2211
**Вариант:** №6 (Подсчет гласных букв)

## Описание
Программа запрашивает у пользователя строку на русском языке и выводит количество содержащихся в ней гласных букв (без учета регистра).

## Структура проекта
* `src/ConsoleApp/` — Исходный код программы на C#.
* `.gitignore` — Список исключений для Git.
* `README.md` — Описание проекта.

## Инструкция по запуску
1. Склонируйте репозиторий.
2. Перейдите в папку `src/ConsoleApp`.
3. Выполните команду `dotnet run`.
