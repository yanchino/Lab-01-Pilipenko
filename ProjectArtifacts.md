# Перечень артефактов проекта Lab-01-Pilipenko

## 1. Исходный код (C#)
- **Program.cs** — Точка входа. Управляет логикой вызова модулей.
- **src/ConsoleApp/Modules/InputModule.cs** — Отвечает за чтение строк из консоли.
- **src/ConsoleApp/Modules/ValidationModule.cs** — Содержит логику проверки строки (на пустоту).
- **src/ConsoleApp/Modules/ProcessingModule.cs** — Логика подсчета гласных и вывод результата.

- **ProjectArtifacts.md** — Текущий перечень материалов.
- **ProjectProtocols.md** — Журнал изменений и действий.
